Setting up Kali Linux Lab

By Sina | Published on June 18, 2026 | Tags: #kali #lab #docker #pentest

Starting fresh with a penetration testing lab setup. This post documents the initial Kali Linux environment configuration for CEH practice.

Lab Architecture

  • Kali Linux as primary OS / VM
  • Docker for vulnerable targets (DVWA, WebGoat, Juice Shop)
  • VirtualBox for additional isolated environments
  • Network isolation for safe testing

Tools Installed

- nmap (network scanning) - Burp Suite Community (web testing) - sqlmap (SQL injection) - metasploit (exploitation) - wireshark (packet analysis) - aircrack-ng (wireless)

Docker Targets

Running vulnerable applications in containers for safe practice:

docker run -p 80:80 vulnerables/web-dvwa docker run -p 8080:8080 webgoat/goatandwolf docker run -p 3000:3000 bkimminich/juice-shop

Next Steps

  • Configure Burp Suite for HTTPS interception
  • Set up logging and monitoring
  • Document discovered vulnerabilities
  • Practice SQL injection and XSS payloads

Lab is operational and ready for practice!
